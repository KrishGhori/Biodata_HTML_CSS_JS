# Biodata_HTML_CSS_JS

#  Biodata Web Page (HTML & CSS)

A modern, responsive **Biodata / Resume web page** built using **pure HTML and CSS**.  
This project is suitable for **college assignments**, **resume presentation**, and **GitHub portfolios**.

---

##  Features

-  Modern card-based UI
-  Automatic Dark Mode (system-based)
-  Print-friendly resume layout
-  Easy PDF export using browser print
-  Fully responsive design
-  College assignment ready
-  No external libraries used

---

##  Technologies Used

- **HTML5**
- **CSS3**
- Media Queries (Responsive + Print + Dark Mode)

##  How to Use

1. Download or clone this repository
2. Open `biodata.html` in any modern web browser
3. To generate PDF:
   - Press `Ctrl + P`
   - Select **Save as PDF**
   - Enable **Background graphics**
